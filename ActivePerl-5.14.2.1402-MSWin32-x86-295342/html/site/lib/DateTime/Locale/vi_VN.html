NAME

DateTime::Locale::vi_VN

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'vi_VN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Vietnamese Vietnam.
