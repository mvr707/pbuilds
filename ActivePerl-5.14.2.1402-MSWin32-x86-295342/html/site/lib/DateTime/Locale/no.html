NAME

DateTime::Locale::no

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'no' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Norwegian.

DATA

This locale inherits from the the DateTime::Locale::nb manpage locale.
