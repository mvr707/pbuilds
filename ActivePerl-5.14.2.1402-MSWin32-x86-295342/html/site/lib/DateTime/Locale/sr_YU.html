NAME

DateTime::Locale::sr_YU

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'sr_YU' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Serbian.

This locale inherits from the the DateTime::Locale::sr_Cyrl_RS manpage locale.
