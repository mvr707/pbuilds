NAME

DateTime::Locale::nn

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'nn' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Norwegian Nynorsk.

DATA

This locale inherits from the the DateTime::Locale::root manpage locale.
