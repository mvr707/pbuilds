NAME

DateTime::Locale::es_HN

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'es_HN' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Spanish Honduras.
