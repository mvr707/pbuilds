NAME

DateTime::Locale::ha_Arab_SD

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'ha_Arab_SD' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Hausa Arabic Sudan.
