NAME

DateTime::Locale::is_IS

SYNOPSIS

  use DateTime;

  my $dt = DateTime->now( locale => 'is_IS' );
  print $dt->month_name();

DESCRIPTION

This is the DateTime locale package for Icelandic Iceland.

This locale inherits from the the DateTime::Locale::is manpage locale.
