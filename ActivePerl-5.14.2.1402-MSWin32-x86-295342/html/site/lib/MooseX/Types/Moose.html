NAME

MooseX::Types::Moose - Type exports that match the types shipped with the Moose manpage

VERSION

version 0.35

SYNOPSIS

  package Foo;
  use Moose;
  use MooseX::Types::Moose qw( ArrayRef Int Str );
  use Carp qw( croak );

  has 'name',
    is  => 'rw',
    isa => Str;

  has 'ids',
    is  => 'rw',
    isa => ArrayRef[Int];

  sub add {
      my ($self, $x, $y) = @_;
      croak 'First arg not an Int'  unless is_Int($x);
      croak 'Second arg not an Int' unless is_Int($y);
      return $x + $y;
  }

  1;

DESCRIPTION

This package contains a virtual library for the MooseX::Types manpage that is able to export all types known to the Moose manpage. See the MooseX::Types manpage for general usage information.

METHODS

type_storage

Overrides the MooseX::Types::Base manpage' type_storage to provide a hash reference containing all built-in the Moose manpage types.

SEE ALSO

the MooseX::Types::Moose manpage, the Moose manpage, the Moose::Util::TypeConstraints manpage
