NAME

MooseX::Types::CheckedUtilExports - Wrap the Moose::Util::TypeConstraints manpage to be safer for the MooseX::Types manpage

VERSION

version 0.35

DESCRIPTION

Prevents errors like:

    subtype Foo =>
    ...

Which should be written as:

    subtype Foo,
    ...

When using the MooseX::Types manpage. Exported by that module.

Exports checked versions of the following subs:

type subtype maybe_type duck_type enum coerce from as

While class_type and role_type will also register the type in the library.

From the Moose::Util::TypeConstraints manpage. See that module for syntax.

SEE ALSO

the MooseX::Types manpage
