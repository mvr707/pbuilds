NAME

dmake - A utility to manage software builds

DESCRIPTION

The dmake PPM package installs the dmake utility into the Perl tree at site/lib/auto/dmake. It also creates a link to dmake.exe in the site/bin diretory so that it can be found on the PATH.

HISTORY

The dmake files installed into site/lib/auto/dmake have been extracted from the authors/id/S/SH/SHAY/dmake-4.11-20080107-SHAY.zip file on CPAN.
